단일 인터페이스 S-행렬 시뮬레이터

Air–Film 단일 인터페이스에 대해, 복소 굴절률 n + i k 를 갖는 두 매질 사이의 scattering matrix S = [[S11, S12], [S21, S22] ]를 계산합니다.
현재는 n, k를 에너지에 대해 상수로 두고, 에너지를 x축으로만 두어 |Sij|와 arg Sij가 올바르게 계산되는지 확인하는 용도의 페이지입니다. (나중에 n(E), k(E)를 넣으면 진짜 에너지 의존 S(E)로 확장 가능)

설정값 입력

※ 현재는 n1, k1, n2, k2를 에너지에 대해 상수로 두었습니다. 따라서 S(E)는 에너지에 대해 상수이고, 그래프는 수평선으로 나오는 것이 정상입니다. 이후 n(E), k(E) 분산을 넣을 경우 이 틀을 그대로 사용하여 S(E)를 확장할 수 있습니다.
※ S-행렬 정의: a1, a2를 각각 좌/우에서 입사하는 파, b1, b2를 각각 좌/우로 반사/투과되어 나가는 파라고 하면,
[ b1 ] = S [ a1 ] = [[S11, S12], [S21, S22]] [ a1 ] 입니다.
